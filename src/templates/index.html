How to use

You should read the README.md before starting out

Params [GET]

  • /api/

    • Returns all storms' data in JSON format.

    • id

      • Returns storm by depression id (i.e. 12L.)

    • name

      • Returns storm by name (i.e. Iota.) Can be lowercase or uppercase.

    • basin

      • Returns all storms in a basin (i.e. ATL.) Can be lowercase or uppercase.

    Responses

    Name Details
    last-updated Date string (HH:MM:SS) of when the API was last updated.
    storms Returns a list of dictionaries for each active storm.
    id Depression of an active storm. Contains a dictionary of the values (basin, date, latitude, longitude, name, pressure, time, vmax). Found in storms response.

    Example url to get Tropical Storm Cristobal (http://127.0.0.1:5000/api/?name=Cristobal)
